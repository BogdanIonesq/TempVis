# TempVis
### Overview
TempVis is a simple Python project which extracts temperatures data for all European coutries from [accuweather](https://www.accuweather.com) and draws a choropleth map.

![alt text](https://github.com/BogdanIonesq/TempVis/blob/master/mapExample.png)
### Motivation
My growing passion for data science.
### Usage
#### Dependencies
* [python3](https://www.python.org/)
* [requests](http://docs.python-requests.org/en/master/)
* [pandas](https://pandas.pydata.org)
* [folium](https://github.com/python-visualization/folium)
#### Running
Go to the project directory and run
`$ python3 TempVis.py`
The map is saved in the same directory as `map.html`.
